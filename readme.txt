利用 WPScan 檢查 WordPress 外掛的安全性。

== Description ==

WordPress plugins security scanner with WPScan.

== Installation ==

1. 下載後將外掛放至目錄 /wp-content/plugins/ 下。
1. 在外掛清單中啟用本外掛。
1. 先註冊 WPScan API https://wpscan.com/api 取得 token。
1. 請至『設定』填入 WPSCAN API token。
1. 到『外掛』，找『掃描外掛安全性』進入。
